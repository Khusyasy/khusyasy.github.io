## Link Shortener v2 - React JS

[<img src="images/react_link_shortener.png"/>](/images/react_link_shortener.png)

- [Link to website](http://sla.khusyasy.xyz/)
- [Link to repository](https://github.com/Khusyasy/shortlink-app)

**Project description:** This is a website where you can shorten your link, and use it and share shortened long links, because shortened links is easier to share.

### Features

- Shorten your link in just 1 click.
- Register and Login to your account and view how many times your link has been clicked.

### Objectives

I wanted to learn the MERN Stack, so i have to combine what i have learned about React JS and Express JS, i also learn how to do CRUD with MongoDB databases using Mongoose, and using JWT for authentication.

### Techonologies used

- MongoDB
- Mongoose
- Express JS
- React JS
- React Router
- Node JS
- JWT
