Link Shortener - Express JS

Project description: This is a website where you can shorten your link, and use it and share shortened long links, because shortened links is easier to share.

Features

  • Shorten your link in just 1 click.
  • Login to see how many times your link has been clicked.

Objectives

I wanted to learn about using Express JS on how to do CRUD with MYSQL databases and using session properly, and learn more about hashing, and to be more familiar working with Node JS in general.

Techonologies used

  • Node JS
  • Express JS
  • Pug
  • Mysql2
